Interactive Website with Databases (UI/UX)

In this project I was able to create an interactive website with a login/logout feature that allows the admin to input information into the existing database I created. With the help of html, css, javascript, and php, I was able to create a music catalog for the famous singer Doja Cat which allows me to keep track her music releases and music catalog and manipulate them to my liking. I was also able to add a filter element which enables the user to filter her songs depending on the category the user wants to choose. This enhances the user experience allowing for a gentle website navigation.
