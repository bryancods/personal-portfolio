Multiplayer Video Game (Using Ocaml)

    Made with:

Using Ocaml, my group mates and I were able to create a 3000 line of code multiplayer video game similar to the famous Ludo game. The goal is to make a player go through the board wihtout getting captured. We implemented some rules (which show up at the beginning of the game) that makes the game more fun such as the player has to roll two 6's in order to get their piece back when they get captured. The overall aim was to use Ocaml to create an interactive game that is fun and easy to play.
